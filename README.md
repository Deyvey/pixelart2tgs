# pixelart2tgs

Simple .gif to .tgs converter cli utility.

![Source GIF](/images/ralsei.gif)
![Tg demonstation](/images/tg.gif)

## Usage examples

``` console
$ pixelart2tgs -i input.gif
input.gif -> input.tgs
```

``` console
$ pixelart2tgs -i first.gif -i second.gif sticker.tgs -y
first.gif -> first.tgs
second.gif -> "sticker.tgs
```
